# Four Worlds Reference Mapping

## Status

Reference only.

Not machine canon.

Use as comparative guidance, not as proof.

## Four worlds reference

Common Kabbalistic reading:

- `Atzilut` = emanation / near-undivided source
- `Beriah` = creation
- `Yetzirah` = formed and living structure
- `Assiyah` = action / embodiment / actualization

## Current packet mapping

| Reference world | Packet layer | Reason |
|---|---|---|
| `Atzilut` | `L1 CHAOS` | near-undivided source, primary pressure, pre-separated state |
| `Beriah` | `L2 FORMATION` | creation and formation of recognizable form |
| `Yetzirah` | `L3 MARKET` | already-formed and living structures that persist, compete, and move |
| `Assiyah` | `L4 DEMAND` | action, embodiment, manifested choice, outward consequence |

## Why this mapping is useful

It matches the machine unusually well:

- `L1` is source-like, not yet fully separated
- `L2` is where form is actually created
- `L3` is where form is alive
- `L4` is where one form becomes actual outward action

## Important caution

This mapping is not used as a mystical justification.

It is useful because:

- it offers a multi-level manifestation reference
- it helps distinguish creation from living form
- it helps distinguish living form from actualized action

## Strongest insight for `L4`

`Assiyah` suggests that `L4` should be:

- high-definition
- actualizing
- consequence-bearing
- selective

Meaning:

- not all market forms pass through
- one form or a very small set becomes real
- the act of manifestation leaves a residue back into the cycle
