# CUDA L2 Neural Boundary Stand

Это тестовый GPU-стенд для `L2 neural boundary`.

Он не пытается быть каноническим `L2`.
Он нужен, чтобы:

- проверить, что `L2` вообще можно перенести на GPU как локальную активационную сетку
- сохранить те же 4 типа узлов:
  - `OBSERVE`
  - `CHOOSE`
  - `ENCODE`
  - `RUNTIME`
- увидеть, как ведёт себя большой `L2` на видеокарте

## Что здесь важно

- размер `L2` выводится из `L1 ring` и `L1 CW`
- `PU_max` выводится из размера `L2`
- шаг считается на GPU
- хост только запускает тики и печатает checkpoints

## Запуск

```bash
make
./l2_neural
./l2_neural 4096 256 160 12345
./l2_neural 8192 1024 160 12345
```

Аргументы:

1. `l1_ring`
2. `l1_cw`
3. `ticks`
4. `seed`
