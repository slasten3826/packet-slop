# Lua Eva.00 Multi-L3 Cost Stand

Этот стенд проверяет гипотезу:

- если `L3` не один, а несколько,
- цена жизни формы в каждом `L3` падает как `1 / l3_count`,
- формы при этом разные,
- и интересно смотреть, растёт ли разнообразие и lifetime.

## Что здесь происходит

1. `Eva.00 encode core` живёт поверх `L1 chaos`.
2. Успешные процессы уходят не в один `L3`, а маршрутизируются в один из нескольких независимых `L3`.
3. В каждом `L3` burn формы уменьшается по закону:

`burn = base_burn / l3_count`

4. Затем сравниваются:

- сколько форм живёт одновременно
- насколько карманы `L3` похожи или отличаются
- меняется ли mode-distribution

## Запуск

```bash
lua main.lua
lua main.lua 128 256 12345 3 72
```

Аргументы:

1. `ticks`
2. `process_count`
3. `seed`
4. `l3_count`
5. `l3_length`
