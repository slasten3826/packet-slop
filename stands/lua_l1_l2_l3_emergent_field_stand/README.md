# lua_l1_l2_l3_emergent_field_stand

First rough `L3` field stand where:

- `L2` emits only encoded seeds
- `L3` does not receive predefined modes
- `CYCLE / LOGIC / RUNTIME` emerge from behavior over time

This stand exists to test the hypothesis that upper `L3` operators should be
born from field dynamics, not assigned directly at `L2 -> L3` handoff.
