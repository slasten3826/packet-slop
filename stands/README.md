# stands

This directory contains executable research stands.

- `lua_crazy_torus_stand`
  CPU-transparent baseline for the L1 chaos law.
- `lua_l2_boundary_stand`
  first executable hex-boundary sandbox for the L2 hypothesis.
- `lua_l2_neural_boundary_stand`
  first neural hex-boundary sandbox for the L2 hypothesis.
- `cuda_crazy_t3_stand`
  CUDA stand for the GPU body of the same law.

These stands are not product code.
They exist to test transfer, parity, scaling, stress, and runtime asymmetry.
