# Lua Eva.00 Encode Core + L3 Stand

Это связанный CPU-стенд:

- `Eva.00 encode core` поверх `L1 chaos`
- текущий `L3` как life-space из:
  - `RUNTIME`
  - `CYCLE`
  - `LOGIC`
  - `MANIFEST`

## Что здесь происходит

1. Много `EncodeProcess` живут поверх `L1`.
2. Они тянут `CHAOS`, собирают `CALM` и иногда успешно доходят до `RUNTIME`.
3. В момент успеха процесс не просто умирает, а эмитит crystal в `L3`.
4. `L3` принимает эти crystals и дальше живёт по текущей локальной топологической физике.

## Что это проверяет

- даёт ли `Eva.00 encode core` осмысленный поток форм в `L3`
- какие `L3 modes` реально рождаются из успешных encode-процессов
- как долго эти формы живут
- уходит ли `L3` в насыщение или держит жизнь
