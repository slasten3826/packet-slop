# lua_l1_l2_l3_transition_stand

First rough stand for the transition hypothesis:

- `L2` contains only `OBSERVE / CHOOSE / ENCODE`
- `L3` contains only `CYCLE / LOGIC / RUNTIME`
- transition happens through `encoded crystal + PU_stock`

This stand exists to test the idea that there is no hard operator-wall between
`FORMATION` and `MARKET`, only a phase transition of form.
